GyaanVault — study material folder
==================================

Drop your PDFs here. Anything in this folder is served on the live site at:
    /study-material/<your-file-name>

Then link it from src/data/notes.js using the shortcut:
    notes: sm("your-file-name.pdf")

Naming tip: start with the course code and use dashes, e.g.
    DS-502-Computational-Mathematics-Notes.pdf

Full step-by-step instructions are in GYAANVAULT-GUIDE.md at the project root.
